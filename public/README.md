# Assets Directory

## Required Assets

To complete your magical bookstore experience, add these files:

### 1. Background Image
- **File**: `magic-bookstore.jpg`
- **Location**: Place directly in this `/public` folder
- **Description**: Your magical bookstore background image
- **Recommended**: High-resolution image (1920x1080 or larger)

### 2. Background Music (Optional)
- **File**: `lofi.mp3`
- **Location**: Place in `/public/audio/` folder
- **Description**: Chill lofi music or any ambient background music
- **Format**: MP3 audio file

## How to Add Files

In Cursor:
1. Drag and drop your image file into the `/public` folder
2. Drag and drop your music file into the `/public/audio` folder

Or use terminal:
```bash
# Copy your files
cp /path/to/your/image.jpg public/magic-bookstore.jpg
cp /path/to/your/music.mp3 public/audio/lofi.mp3
```
